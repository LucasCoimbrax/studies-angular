# studies-angular

Passos

1º Verificar se o node esta instalado

    node --version

2º Instalar o angular na máquina

    npm install -g @angular/cli

3º Iniciar um novo projeto angular

    ng new nome-do-projeto      -> ng new primeiro-projeto

4º Rodar o projeto

    npm run start

5º Para criar um novo componente dentro do projeto na pasta app

    ng generate component components/home
    ng g c components/home

6º Para criar um novo service dentro da pasta app

    ng generate service services/enviaFormulario
    ng g s services/enviaFormulario
